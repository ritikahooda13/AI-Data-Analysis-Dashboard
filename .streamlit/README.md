# 📊 AI Automated Data Analysis Dashboard

An automated, data-agnostic exploration and analytical visualization dashboard built with **Python** and **Streamlit** that enables users to upload datasets and instantly generate core mathematical metrics and interactive charts.

---

## 🔍 Core Capabilities
* **Dynamic File Ingestion**: Seamless ingestion engine supporting multi-variant CSV schema integration.
* **Automated Data Auditing**: Live execution of shape dimensions, structural metrics, and comprehensive tracking of missing parameters.
* **Smart Filtering Matrix**: Dynamic numerical range mapping and categorical query parameters based on detected structural columns.
* **Statistical Exploration Profile**: Statistical aggregations including numerical variance tracking, skew metrics, and automated correlation matrices.
* **Advanced Chart Render Engine**: Dynamic visualization components for scatter analysis, value distributions, and correlation maps using Seaborn.

---

## 🛠️ Tech Stack & Architecture
* **Core Runtime Environment**: Python 3.10+
* **Data Synthesis Engines**: Pandas, NumPy, SciPy
* **Visual Graph Rendering**: Matplotlib, Seaborn
* **Dynamic Frontend Layer**: Streamlit Framework
