# AI Automated Data Analysis Dashboard

A production-grade, data-agnostic automated analysis engine engineered using Python and Streamlit. This application dynamically parses structural datasets to execute advanced statistical profiling, relational filters, and interactive graphical exploration pipelines.

## 🚀 Core Features
* **Automated Data Auditing:** Intelligent schema parsing capable of dynamic file extraction, dimensional logging, and field completeness analytics.
* **Dynamic Multidimensional Queries:** Responsive filtering rules panel using custom categorical filters and numerical range sliders.
* **Statistical Profiling & Visualizations:** Continuous variable distribution metrics, covariance summaries, automated bivariate scatter-plots, and relational correlation heatmaps.
* **Flexible Architecture:** Highly adaptable data-agnostic engine designed to process varying external structural CSV datasets without crashing.

## 🛠️ Technology Stack
* **Framework:** Streamlit
* **Data Processing:** Pandas, NumPy, SciPy
* **Graphics & Analytics:** Matplotlib, Seaborn
